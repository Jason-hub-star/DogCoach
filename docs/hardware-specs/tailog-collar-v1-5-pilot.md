# TailLog Smart Collar V1.5 (Pilot+)

## 1. 목표
- MVP 이후 실사용 안정화
- 100가구 내외 파일럿 운영 대응
- 양산 전 리스크 제거

## 2. 필수 기능
- MVP 전 기능 유지
- 이벤트 정확도 개선(오탐 감소)
- OTA 롤백/복구
- 착용감 개선(밴드/하우징)
- 진단 로그 수집(배터리/연결/센서 이상)

## 3. 목표 스펙
- 무선: BLE 5.2
- 센서: 6축 IMU + MEMS 마이크 (+온도 선택)
- 배터리: 10~14일
- 무게: 25~35g
- 방수: IP67~IP68
- 충전: 자석식 포고핀(내구 강화)

## 4. 추가 항목 (선택)
- 온도 센서
- 저전력 진동/LED 피드백
- 착용 감지 개선 알고리즘

## 5. 인증/규격
- KC 유지/확장 시험
- Bluetooth SIG 유지
- 양산용 인증 문서 패키지 준비

## 6. 단가 목표
- 양산 목표: 3.0만~3.8만원/개
- 파일럿 허용: 4.0만원대 초반

## 7. 완료 기준 (Definition of Done)
- 파일럿 4주 이상 안정 운영
- 배터리 10일 이상 평균 달성
- OTA 실패 복구율 100%
- 양산 BOM/공정/검사 항목 확정
