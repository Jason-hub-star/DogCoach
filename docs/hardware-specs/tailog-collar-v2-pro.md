# TailLog Smart Collar V2 (Pro)

## 1. 목표
- 프리미엄 라인 확장
- 위치 추적/안전 기능 강화
- 해외 시장 진출 준비

## 2. 필수 기능
- V1.5 기능 전체
- LTE-M 기반 원격 연결
- GNSS 위치 추적 및 지오펜스
- 분실 모드/긴급 위치 전송
- 원격 펌웨어/설정 관리

## 3. 목표 스펙
- 무선: BLE + LTE-M (GNSS 포함)
- 센서: 6축 IMU + MEMS 마이크 + 온도 (+추가 센서 선택)
- 배터리: 3~7일 (LTE 사용 시나리오 기준)
- 무게: 35~50g
- 방수: IP67~IP68
- 충전: 도크 또는 고내구 포고핀

## 4. 확장 항목 (선택)
- UWB 근거리 정밀 탐색
- 심박/바이오센서
- 멀티벤더 홈 IoT 직접 제어

## 5. 인증/규격
- 국내: KC
- 해외: FCC + CE(RED)
- 필수: Bluetooth SIG Qualification
- 통신사/망 연동 검증 절차 포함

## 6. 단가 목표
- 양산 목표: 5만원+/개 (통신 모듈/안테나 포함)
- 판매 전략: 프리미엄 요금제 번들

## 7. 완료 기준 (Definition of Done)
- 실사용 위치 추적 정확도 KPI 충족
- LTE 연결 안정성/통신비 모델 검증 완료
- FCC/CE 시험 착수 가능한 EVT/DVT 샘플 확보
- 해외 출시용 문서/라벨링 요구사항 반영 완료
