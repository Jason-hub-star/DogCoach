# TailLog Smart Collar V1 (MVP)

## 1. 목표
- 1차 파일럿 검증용 최소 기능 제품
- 핵심 가치: 행동 데이터 수집 + 이상 이벤트 알림
- 원칙: 저전력, 저단가, 빠른 개발

## 2. 필수 기능
- 활동/휴식 시간 기록
- 이상 이벤트 감지(과활동, 짖음 패턴 증가)
- BLE 앱 동기화
- 배터리 잔량/저전력 알림
- 기본 OTA 펌웨어 업데이트

## 3. 목표 스펙
- 무선: BLE 5.2 (단일)
- 센서: 6축 IMU + MEMS 마이크
- 배터리: 최소 7일, 목표 10일
- 무게: 25~35g
- 방수: IP67 이상
- 충전: 자석식 포고핀

## 4. 제외 항목 (V1 미포함)
- LTE/GPS 실시간 추적
- 심박/고급 바이오센서
- 자동급식기 직접 제어(앱 알림 중심)

## 5. 인증/규격
- 필수: KC
- 필수: Bluetooth SIG Qualification
- 전략: 무선 인증 모듈 우선 적용

## 6. 단가 목표
- 양산 목표: 2.8만~3.5만원/개
- 초기 파일럿 허용: 4.0만~4.5만원/개

## 7. 완료 기준 (Definition of Done)
- 배터리 실사용 7일 이상 달성
- BLE 재연결 안정성 검증 완료
- 방수/낙하 기본 테스트 통과
- KC 인증 착수 가능한 시제품 BOM 확정
